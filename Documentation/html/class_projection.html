Creates an intensity projection of the Volume using the specified compare function, output filename, and the range of images to be considered in the projection (min_z and max_z).

Defines the IP method for the Projection class, taking the compare function, output name, and min/max z values as parameters.

Parameters
compare_functionThe name of the function to use for intensity comparison (either "max" or "min").
output_nameThe filename to use for the output projection image.
min_zThe index of the first image to be considered in the projection.
max_zThe index of the last image to be considered in the projection.
flagA string flag indicating the direction of the projection (either "xy" or "xz").
compare_functionA C-style string representing the type of projection operation to perform (max, min, mean, or median).
output_nameA C-style string representing the name of the output file to save the projection to.
min_zAn integer representing the minimum z-value of the thin slab to consider.
max_zAn integer representing the maximum z-value of the thin slab to consider.
flagA std::string object that, when set to "test", tests the output against predetermined values rather than saving it to a file.
