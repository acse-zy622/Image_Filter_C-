◆ ApplyGrayScale()

void Filter::ApplyGrayScale ( )

Applies grayscale to an image.

Calculates the average of each RGB value within the pixel and stores it as an unsigned 8-bit integer in the output pixel. If the image has an alpha channel, it stores the alpha channel in the second channel of the grayscale image.

void Filter::gaussian_blur ( )

Applies Gaussian blur to the input image.

This function uses a Gaussian kernel to apply a Gaussian blur to the input image. The Filter class instantiation includes the kernel size and standard deviation/sigma. The appearance of the final image will be smoother, with less noise and detail.

◆ gaussian_blur_3d()

void Filter::gaussian_blur_3d ( )

Applies a 3D Gaussian blur filter to the input image data for each channel. The blur is computed using a Gaussian kernel with the specified size and standard deviation. The input data is assumed to be stored in row-major order with interleaved channels.

Parameters
dataPointer to the input image data. The size of the data buffer must be at least w * h * d * c bytes.
wWidth of the input image in pixels.
hHeight of the input image in pixels.
dDepth of the input image in pixels.
cNumber of channels in the input image.
kernel_sizeSize of the Gaussian kernel in each dimension (e.g., 3 for a 3x3x3 kernel).
sigmaStandard deviation of the Gaussian kernel in each dimension.

◆ Prewitt()

void Filter::Prewitt ( )

Applies the Prewitt edge detection filter to the image.

This function utilizes the Prewitt operator to determine the gradient magnitude for each pixel and applies the Prewitt edge detection filter. The operator is a pair of 3x3 convolution kernels, for the X and Y direction each. The resulting image emphasizes the edges within the input image.

void Filter::Sobel ( )

Applies the Sobel edge detection filter to the image.
