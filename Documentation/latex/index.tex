The purpose of this project is to design and implement a C++ program that applies a range of filters and orthographic projections to input 2D images or 3D data volumes. This program includes eight 2D image filters, two 3D data volume filters, and 3D image projections and slices.\hypertarget{index_autotoc_md2}{}\doxysection{Installation}\label{index_autotoc_md2}
To use this library, you will need to have the following dependency installed\+: STB \mbox{\hyperlink{class_image}{Image}} for reading and writing image files.\hypertarget{index_autotoc_md3}{}\doxysection{Usage}\label{index_autotoc_md3}
To use the library, you can include the necessary header files in your code and link against the library.
